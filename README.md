# Документация (обоснование выбора UI)
День 12.
Выбор: CustomTkinter (надстройка над стандартным tkinter).
Обоснование выбора:
В 2026 году, несмотря на наличие мощных фреймворков вроде PyQt6 или Kivy, для разработки десктопных приложений на Python, ориентированных на скорость разработки и простоту поддержки, CustomTkinter является оптимальным выбором.
Причины:
Современный дизайн "из коробки": Стандартный Tkinter выглядит устаревшим. CustomTkinter предоставляет темную тему, скругленные углы и современные кнопки без необходимости писать сотни строк CSS-подобного кода.
Нативность и легкость: В отличие от Electron или веб-интерфейсов, это нативное Python-приложение, которое работает быстро и не требует установки дополнительных сред выполнения (как Node.js).
Простота интеграции графиков: Библиотека отлично дружит с matplotlib, позволяя встраивать интерактивные графики прямо в виджеты.
Масштабируемость: Архитектура позволяет легко создавать отдельные классы окон для каждой утилиты, что соответствует требованию модульного расширения SuperApp.
